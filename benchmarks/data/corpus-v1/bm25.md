# BM25

BM25 (Best Matching 25) is a ranking function used by search engines to estimate
the relevance of documents to a query. It was developed by Stephen Robertson
and Karen Spärck Jones and published in the Okapi system in the 1990s.

## Formula

BM25 extends TF-IDF with a saturation function on term frequency and a length
normalization penalty. The score for a document `D` against a query containing
terms `q_1 ... q_n` is:

```
BM25(D, Q) = Σ IDF(q_i) · (f(q_i, D) · (k1 + 1)) / (f(q_i, D) + k1 · (1 - b + b · |D| / avgdl))
```

where `k1` controls term-frequency saturation (typical value 1.2–2.0), `b`
controls length normalization (typical 0.75), and `avgdl` is the average
document length in the corpus.

## Why it wins over naive TF-IDF

Raw term frequency scales linearly — a document that mentions a term 100 times
would score 100× a document that mentions it once. BM25's saturation makes the
marginal score of each additional occurrence diminish, which better matches how
humans judge relevance.
